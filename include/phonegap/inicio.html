1ª Criar o projeto

Após entrar na pasta aonde se quer criar o projeto, digitar o comando, abaixo para baixar o projeto baseado nas abas flutuantes.
OBS:Existem outros tipos de projetos. 
$ ionic start myApp tabs
Após baixar , basta entrar na pasta do projeto, no exemplo a pasta do projeto é:myApp 
$ cd ./myApp/
Agora executaremos o comando para rodar a aplicação 
$ ionic serve
Deve demorar um pouco, mas o retorno deve ser algo parecido com isso: 
[INFO] Development server running!
       
        Local: http://localhost:8100
        External: http://192.168.0.17:8100
        DevApp: myApp@8100 on desk
        
        Use Ctrl+C to quit this process
 
 [INFO] Browser window opened to http://localhost:8100!
Se chegarmos até aqui, basta abrir o navegador com a url: http://localhost:8100
