Bloquear Elemento

Aqui vai uma opção interessante, sempre que você desabilita o componente, o mesmo não é submitido no POST, abaixo uma maneira de bloquear no form entretando submeter a informação contida no mesmo! 
CSS
.desabilitado {background:#A0A0A0}

JS
$(".desabilitar").bind("keyup keydown click change", function(){  return false; });
$(".desabilitar").attr("disabled","disabled");	

HTML
<input type="text" name="nm_titular" value="Roberto" class="form desabilitar" />

HTML
<div id="teste">
	<img src="images/maparea_02.jpg" class="testeImagem">
</div>
CSS
#teste{
    width:150px;height: 150px;
    border:2px solid red;
    overflow: hidden;
}
#teste > img.testeImagem {
    position: relative;
    top: 50%;
    left: 50%;
    margin-right: -50%;
    transform: translate(-50%, -50%);
