É um link simbólico, isto é, um alias ou atalho para um programa ou arquivo. É um arquivo especial existente no FileSystem e apontando para outro arquivo ou diretório. Se você acessar o link simbólico de um aplicativo, ele parece transparente para o aplicativo e você realmente acessará o arquivo ou o diretório ao qual o link simbólico está apontando. Embora um link simbólico apareça com permissões de arquivos e propriedades de usuários / grupos, os direitos de acesso são determinados apenas pelas permissões de destino e pelos proprietários de usuários / grupos. Criar o link: 
$ ln -s
Deletar o link: 
$ rm [name of the symlink]
Caso de uso, montar um pendrive/hd e criar um link na pasta /var/www com nome de 'dados' 
$ cd /var/www
$ ln -s /media/beto/DADOS/ dados
