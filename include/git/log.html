LISTAR logs

$ git log

LISTAR apenas os arquivos modificados nos commits no log

$ git log --name-only

Verificar os arquivos que foram commitados

$ git log --name-status

ou de um commit especifico *SEU HASH

$ git show --pretty="" --name-only bd61ad98*

Verificar qual o commit de determinado arquivo

$ git log --follow ARQUIVO

Verificar o que foi mudado no arquivo, no caso testei somente o primeiro comando.

$ git diff HEAD^^ myfile
ou 
$ git diff --cached
ou
$ git diff --staged

Quando se deseja voltar a versão do arquivo para o ultimo commit

$ git checkout HEAD -- my-file.txt

Ordebar o git status por datetime:

$ git status -s | while read mode file; do echo $mode $(date --reference=$file +"%Y-%m-%d %H:%M:%S") $file; done

ou 

git status -s | while read mode file; do echo $mode $(stat -c %y $file) $file; done|sort -k1.4

Lista o historico de commits de forma colorida, (Jorge Titoneli)

$ git log --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit
