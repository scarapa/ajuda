COMMIT

Adicionar TODOS os arquivos

$ git add .

Adicionar os arquivos INDIVIDUALMENTE

$ git add CAMINHO/ARQUIVO

Adiciona os arquivos por pasta no exemplo do comando é a pasta include

$ git add include/*

Commitar arquivos DELETADOS

$ git add --all
$ git add -A
A diferença do $ git add . é que o --all também atualiza os ficheiros removidos. O add . não faz nenhuma ação com ficheiros removidos.

Commitar TODOS os arquivos e incluir a mensagem de commit ao mesmo tempo

$ git commit -am "mensagem"

Alterar MENSAGEM antes do PUSH

O $ git commit --amend vai abrir seu editor, com o conteúdo da mensagem do último commit e você pode editar tranquilamente.
$ git commit --amend

Listar arquivos do commit

Para listar TODOS os arquivos do COMMIT é necessário saber o hash do commit, tal dado consegue-se na interface web.

SINTAXE: 
$ git show --name-only HASH
EXEMPLO: 
$ git show --name-only 11c19cb2
