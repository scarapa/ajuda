Criar Projeto Localmente com ANGULAR

$ ionic start nome_projeto blank --type=angular

Estrutura

Criar pagina

$ ionic g page nome_pagina

Criar Provider/Service

$ ionic g service nome_provider

Banco de Dados

Salvar pelo TERMINAL

$ adb -d shell "run-as [PACOTE] cat [PASTA DEVICE]" > [PASTA MAQUINA LOCAL]
Conforme exemplo abaixo: 
$ adb -d shell "run-as br.tche.emater.expointer cat databases/expointer.db" > ~/Documentos/android.db

Salvar pelo ANDROID STUDIO

Pode-se salvar o banco de dados da aplicação , para isso abrimos o Android Studio e exploramos o device da seguinte forma: Menu >> View >> Tool Windows >> Device Explorer

basta seguir o caminho da apliacao no explorador: data >> data >> nome do pacote >> databases >> nome da base

Leitura - SQLITE

Para abrir e testar querys pode fazer a utilização de ferramente online ou programa local ...
online - sqliteonline
sqliteonline
programa - DB Browser
DB Browser , disponivel para TODOS os sistemas operacionais.

Startar Projeto

Localmente

$ ionic serve

Localmente 2 ambientes

$ ionic serve --labs

Device Android DISPOSITIVO

$ ionic cordova run android --device

Device Android EMULADOR

$ ionic cordova run android --emulator
