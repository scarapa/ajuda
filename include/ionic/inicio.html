Uma vez com NodeJS e o NPM instalados, é necessário instalar 2 módulos globais. 
$ npm install -g ionic cordova
Uma vez com esses dois módulos instalados, usaremos o gerador do Ionic CLI para criar um novo projeto.

Existe 2 formas de iniciar o projeto, pode ser com uma a pagina em branco[1] ou já startando com uma estrutura definida[2]

1. Iniciando projeto em branco

$ ionic start nome_do_projeto blank --type=angular

2. Iiniciando projeto definido

$ ionic start
