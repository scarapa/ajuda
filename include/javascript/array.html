Possibilita testar todos os elementos do seu array. Caso algum dos elementos não passe pela condição definida por você, o retorno será false. 
ES5
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];

function isGreaterThan(age) {
  return age >= 18;
}

ES6
const isGreaterThan = (age) => age > 18;
const greater = ages.every(isGreaterThan);

console.log(greater); // true

FILTRAR

Este método possibilita a criação de filtros de forma rápida e fácil em seus arrays. 
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
// ES5
function startWithS(word) {
  return word.indexOf('S') === 0;
}
ES6
const startWithS = (word) => word.indexOf('S') === 0;
const beersStartWithS = beers.filter(startWithS);
console.log(beersStartWithS); 
// [0: 'San Diego', 1: Saint Bier]

MAPEAR

Este método percorre todos os elementos do array, executando funções para cada elemento e retornando um novo array como resultado. 
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
ES5
function upAge(age) {
  return age + 1;
}

ES6
const upAge = (age) => age + 1;
const newAges = ages.map(upAge);
console.log(newAges); // [0: 21, 1: 26, 2: 20, 3: 22, 4: 43]
> [0: 21, 1: 26, 2: 20, 3: 22, 4: 43]

ALGUM

Este método verifica se ao menos um elemento satisfaz a condição definida por você. 
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
ES5
function hasHeinekenOrSaint(beer) {
  return (beer === 'Saint Bier' || beer === 'Heineken');
}

ES6
const hasHeinekenOrSaint = (beer) => (beer === 'Saint Bier' || beer === 'Heineken');
const heinekenSaint = beers.some(hasHeinekenOrSaint);
console.log(heinekenSaint); // true

REDUZIR

Você pode utilizar o método reduce para alguns casos, sendo um deles para facilitar cálculos.

ES6
const reducerAge = (accumulator, age) => accumulator + age;
const sumAges = ages.reduce(reducerAge);
console.log(sumAges); // 127

Validar/Verificar elemento

Esse metodo verifica se uma determinada condicao é atendida por algum elemento do array

Sintaxe
array.some(callbackfn:(value,index,array)=>{});
  • callbackfn: funcao para verificar determinada condicao
  • value: valor atual
  • indice: chave do valor atual
  • array: manipulado
Exemplo 1
condicao verdadeira 
const numero = [10,20,30,40,50];
const resultado = numeros.some(value,index,array) =>{
	return value > 10;
});
//true
Retorna TRUE porque condição é VERDADEIRA!!!
Exemplo 2
Condicao FALSA! 
const numero = [10,20,30,40,50];
const resultado = numeros.some(value,index,array) =>{
	return value > 80;
});
//false
Retorna FALSE porque condição é FALSA!!!
