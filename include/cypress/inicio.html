1º STARTANDO o projeto

Iniciar o projeto e a criacao do json 
$ npm init -y
OBSERVACAO, o conceito de dependencias de execucao e de desenvolvimento.
Esses conceitos ficam armazenados no package.json (devDependencies/dependencies) Instalar o CYPRESS , um 
$ npm install --save-dev
ou 
$ npm i -D cypress
ou para instalar GLOBALMENTE 
$ npm install cypress -g
OBSERVACAO para DESINSTALAR o CYPRESS GLOBALMENTE 
$ npm uninstall -g cypress
Agora iremos abrir o arquivo (package.json) para editarmos 
$ code package.json
Alteraremos a linha test 
"scripts": {
    "test": "cypress open"
  },

$ npm test
Após a instalacao , executar o Cypress 
$ cypress open

2º ARMAZENAMENTO DOS TESTES

São dos arquivos importantantissimos para que o teste funcione MEU_PROJETO\cypress\support\commands.js
É recomendável que cria uma pasta para isolarmos nossos testes de outros testes, no exemplo, criamos a pasta 01 MEU_PROJETO\cypress\integration\01\NOME_DO_TESTE.JS
