CRIANDO USUARIO REMOTO

MYSQL
CREATE USER 'remoto'@'%' IDENTIFIED BY '***';

GRANT ALL PRIVILEGES ON * . * TO 'remoto'@'%' IDENTIFIED BY '***' 
WITH GRANT OPTION MAX_QUERIES_PER_HOUR 0 MAX_CONNECTIONS_PER_HOUR 0 MAX_UPDATES_PER_HOUR 0 MAX_USER_CONNECTIONS 0 ;

Derrubando Usuarios

Caso haja a necessidade de DERRUBAR USUARIOS por qualquer circustancia há uma funcionalidade
você pode ter que derrubar os usuarios para fazer um export ou import de determinada database por exemplo. 
SQLSERVER
USE [master]

ALTER DATABASE [SISCLAS_DEV]
SET SINGLE_USER --SETA como usuário unico para fechar conexões antes de restaurar backup

    CODIGO ANSISQL

SET MULTI_USER; --volta para multi user depois de restaurar
GO
